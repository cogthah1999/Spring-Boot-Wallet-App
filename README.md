# 💰 Personal Wallet Web App
Ứng dụng quản lý chi tiêu cá nhân viết bằng **Spring Boot** và **SQLite**.

### Tính năng:
- ✅ Quản lý CRUD (Thêm, Xem, Sửa, Xóa) giao dịch.
- ✅ Tìm kiếm giao dịch thông minh.
- ✅ Cảnh báo chi tiêu lớn (Màu đỏ cho khoản > 1tr).
- ✅ Biểu đồ tròn (Chart.js) phân tích tỷ lệ chi tiêu.
- ✅ Tự động tính tổng số dư thực tế.

### Công nghệ sử dụng:
- Java 21, Spring Boot 3, Spring Data JPA, Thymeleaf, SQLite, Chart.js.
